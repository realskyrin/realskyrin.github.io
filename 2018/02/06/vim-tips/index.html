Vim 常用命令

发表于 2018-02-06 | 更新于 2018-12-05 | 评论数：

概述

本文记录一些 vim 的常用命令

多行删除
:1,10d                      //删除1~10行
显示行号
:set num                    //可以在 ~/.vimrc 中设置为永久生效项
修改文本编码格式
:set encoding=utf-8
搜索内容
/aaa                        //从当前位置往后搜索 aaa    
?aaa                        //从当前位置往前搜索 aaa
内容替换
:%s/aaa/bbb/g               //替换全文 aaa 为 bbb
:s/aaa/bbb/                 //替换当前行第一个 aaa 为 bbb
:s/aaa/bbb/g                //替换当前行所有 aaa 为 bbb
:66s/aaa/bbb/g              //替换 66 行 aaa 为 bbb
